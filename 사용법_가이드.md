# 🚀 SNS 트렌드 콘텐츠 반자동화 시스템 - 사용 가이드

100% 무료 구조 | 네이버 API + 해외 RSS + 무료 AI 이미지 + 클로드 프롬프트 자동 조립

---

## 1️⃣ 최초 설치 (한 번만)

### ① 라이브러리 설치
Windows 명령 프롬프트(cmd)에서:
```
pip install requests pyperclip
```
- `pyperclip`이 있으면 프롬프트가 **클립보드에 자동 복사**됩니다 (강력 추천)
- 설치 안 해도 txt 파일로 저장되니 작동은 합니다

### ② 네이버 API 키 무료 발급 (5분, 국내 뉴스용)
1. https://developers.naver.com/apps 접속 → 네이버 로그인
2. **[애플리케이션 등록]** 클릭
3. 애플리케이션 이름: 아무거나 (예: 뉴스수집기)
4. 사용 API: **"검색"** 체크
5. 환경 추가: **WEB 설정** 선택 → 웹 서비스 URL에 `http://localhost` 입력
6. 등록 완료 → **Client ID** 와 **Client Secret** 복사

### ③ 프로그램 첫 실행

> ⚠ **`can't open file ... No such file or directory` 에러가 뜬다면?**
> 명령 프롬프트가 열린 위치(예: `C:\Users\01144`)에 파일이 없어서 생기는 문제입니다.
> **파일이 있는 폴더에서 실행**해야 해요. 가장 쉬운 방법:
>
> 1. 파일 탐색기에서 `sns_trend_system.py`를 저장한 폴더(보통 **다운로드**)를 연다
> 2. 탐색기 **주소창을 클릭**하고 `cmd` 라고 입력 후 Enter
>    → 그 폴더 위치에서 명령 프롬프트가 바로 열립니다
> 3. 이제 아래 명령 실행
>
> 또는 cmd에서 직접 이동해도 됩니다: `cd Downloads` (다운로드 폴더로 이동)
>
> 💡 관리하기 편하게 `C:\sns_system` 같은 전용 폴더를 만들어 파일을 옮겨두는 것을 추천합니다.
> 설정 파일과 이미지가 전부 이 폴더 안에 생성되기 때문이에요.

```
python sns_trend_system.py
```
첫 실행 시 같은 폴더에 설정 파일들이 자동 생성됩니다:

| 파일 | 역할 |
|------|------|
| `config.json` | 네이버 API 키, 허깅페이스 토큰(선택) |
| `channels.json` | 채널 목록·톤앤매너·키워드 (자유 수정) |
| `guidelines.json` | 시스템 가이드라인 (자가 판올림 대상) |
| `rss_sources.json` | 해외 RSS 소스 목록 (자유 추가) |
| `history.json` | 사용한 뉴스 기록 (중복 방지) |

### ④ config.json 열어서 키 입력
메모장으로 열고:
```json
{
  "naver_client_id": "발급받은_Client_ID",
  "naver_client_secret": "발급받은_Secret",
  "hf_token": "",
  "news_per_source": 5
}
```
- `hf_token`은 **비워둬도 됩니다** → Pollinations 무료 서버로 이미지 생성
- 허깅페이스 토큰을 넣으면 FLUX를 먼저 시도하고, 한도 초과 시 자동으로 Pollinations로 전환

## 2️⃣ 매일 사용 흐름 (3분)

```
python sns_trend_system.py
```

1. **채널 선택** — 번호 입력 (a=채널 추가, d=삭제)
2. **핫이슈 리스트 확인** — 국내🇰🇷 + 해외🌏 뉴스가 채널 적합도(⭐)순으로 정렬됨
3. **뉴스 번호 입력** — 해외 기사면 자동으로 트리플 검증 절차가 프롬프트에 포함됨
4. **이미지 자동 생성** — `images/` 폴더에 1080×1080 배경 저장 (텍스트 여백 확보 구도)
5. **클립보드에 프롬프트 자동 복사** → **클로드 앱 열고 Ctrl+V**
6. 클로드가 출력: 블로그 제목 3개+본문 / 카드뉴스 5장 자막+레이아웃 / 릴스 Hook+대본+캡션+해시태그
7. 미리캔버스/캔바에서 이미지+자막 조합 → 발행 🎉

## 3️⃣ 채널 추가/수정하기

**방법 A (프로그램 안에서):** 채널 목록 화면에서 `a` 입력 → 질문에 답하면 자동 등록

**방법 B (파일 직접 수정):** `channels.json`을 메모장으로 열어 수정
```json
{
  "name": "새 채널 이름",
  "tone": "톤앤매너 설명",
  "keywords": ["키워드1", "키워드2"],
  "image_style": "이미지 분위기 (영어)",
  "banned_words": ["금지어1"],
  "naver_queries": ["네이버 검색어1", "검색어2"],
  "rss_categories": ["health"]
}
```
- `keywords`: 뉴스 적합도 점수 계산에 사용 (많을수록 정렬 정확)
- `banned_words`: 의료법/과장광고 리스크 표현 → 프롬프트에 금지어로 전달됨
- `rss_categories`: `rss_sources.json`에 정의된 카테고리명 (`health`, `lifestyle` 기본 제공, 직접 추가 가능)

## 4️⃣ 자가 판올림 (트렌드 상시 반영)

뉴스 선택 화면에서 번호 대신 이렇게 입력:
```
[알고리즘 업데이트]: 요즘 릴스는 3초 안에 결론부터 말하는 게 대세
```
→ `guidelines.json` 버전이 자동으로 올라가고(v1.0→v1.1), **다음 프롬프트부터 이 규칙이 클로드에게 전달**됩니다.

## 5️⃣ ⚠ 꼭 지켜야 할 저작권 원칙

프롬프트에 이미 안전장치가 들어 있지만, 발행 전 직접 확인하세요:
- 기사 문장 **그대로 복사 금지** — 요약 + 출처(매체명) 명시
- 글의 절반 이상은 **내 관점의 해석/인사이트**로 채우기
- 해외 기사 = 번역 전재도 저작권 침해 → 반드시 "재해석 콘텐츠"로만 활용
- 건강 콘텐츠: "치료", "완치", "100% 효과" 같은 표현 금지 (금지어 필터 활용)

## 6️⃣ 문제 해결

| 증상 | 해결 |
|------|------|
| `can't open file ... No such file or directory` | 파일이 있는 폴더에서 실행 (①-③ 참고: 탐색기 주소창에 `cmd` 입력) |
| 국내 뉴스가 안 나옴 | config.json에 네이버 키 입력했는지 확인 |
| 해외 뉴스만 안 나옴 | 방화벽/인터넷 확인, rss_sources.json URL 점검 |
| 이미지 생성 실패 | 잠시 후 재시도 (Pollinations 서버 혼잡 시 발생) |
| 클립보드 복사 안 됨 | `pip install pyperclip` 후 재실행, 또는 prompts 폴더의 txt 파일 사용 |
| 같은 뉴스가 계속 나옴 | 정상입니다 — 선택한 뉴스만 history에 기록됩니다. 강제 초기화는 history.json 삭제 |
